About Go Raffle Me

GoRaffleMe is an open source application dedicated to helping non-profit companies raise the funds that they need to improve our communities.

GoRaffleMe allows organizations to quickly set up raffle drawings. These drawings are managed automatically by the application.

Non-Profits can share links to the raffle event via social media, emails, etc.

Transactions are handled online through PayPal, making GoRaffleMe secure and environmentally friends.

Save Time. Save Trees. Save your Community. GoRaffleMe.
